# 📸 Repository Preview

Add this beautiful preview image to your repository!

## Creating a Social Preview Image

1. **Go to your repository settings:**
   - Click "Settings" tab
   - Scroll to "Social preview"

2. **Upload an image:**
   - Recommended size: 1280 x 640 pixels
   - Format: PNG or JPG

- **Figma** - https://figma.com (professional)
